Qu'est-ce que c'est ?

La technologie pour servir la démocratie !


Demos est une plateforme d'accès aux comptes-rendus officiels des collectivités locales.
L'idée de Demos est de faciliter l'accès et la diffusion des décisions politiques locales.
Demos a pour objectif de réunir les délibérations, procès-verbaux et autres comptes-rendus officiels de toutes les collectivités locales : communes, intercommunalités, départements, régions.

Le projet est né de 2 constats :
  1. En dehors des métropoles régionales, la vie politique locale est mal représentée dans les média.
  2. Les décisions politiques locales sont mal représentées sur le web : l'information est éclatée sur les sites web des différentes collectivités, aucun moteur de recherche n'en propose une vue d'ensemble.

Demos a pour objectif de réunir l'ensemble des décisions politiques locales accessibles sur le web, en vue de fournir ne nouveaux moyens d'accès à cette information pour les citoyens, les journalistes, les chercheurs, etc...

Pour collecter ces informations, Demos parcourt le web et identifie les documents officiels publiés par chaque mairie, intercomunalité, département, région.

Demos est un projet en construction. L'état des lieu aujourd'hui consiste à collecter les procès-verbaux des conseils municipaux des municipalités de France. Les perspectives d'évolutions sont :
  1. collecter les procès-verbaux des conseils intercommunaux, des conseils départementaux et conseil régionaux
  2. structurer l'information collectée, en archivant individuellement chaque des délibération des procès-verbaux collectés
  3. classifier les délibérations par thématique : scolarité, économie, social, infrastructure, agriculture, environnement, etc...
Demos a pour vocation d'utiliser les technologies du web pour servir l'intérêt public.
Le code source est donc libre d'accès sur GitHub.

Les contributions sont les bienvenues !
